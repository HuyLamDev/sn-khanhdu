# Princess Rescue Game

Small Phaser 3 browser game scaffold for a three-stage fairy-tale journey where the princess rescues the prince.

## Stack

- Phaser 3
- TypeScript
- Vite

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Current Phase

Phase 1 provides the project foundation:

- Vite + TypeScript + Phaser setup
- Shared game config and lightweight session systems
- Boot, intro, quiz, maze, runner, win, and lose scenes
- Typed story, question, and stage content modules

Only the intro scene is wired to advance into the quiz stub. The remaining gameplay scenes are placeholders for later phases.
